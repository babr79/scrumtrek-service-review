# Корпоративная ИИ-платформа

← [AI-направление](обзор-ai.md) · [Все услуги](../../каталог-услуг.md)

> Собственная корпоративная ИИ-платформа: мультимодельный доступ, база знаний, AI-ассистенты и enterprise-безопасность — всё в едином контуре с соответствием 152-ФЗ.

## На каком уровне работает

| Уровень | Статус | Описание |
| --- | --- | --- |
| **Стратегический** | ✔ | Даёт инфраструктурную основу для AI-стратегии. Policy Engine и K-токен экономика позволяют управлять AI на уровне компании. Аналитика использования для принятия стратегических решений. |
| **Координационный** | ✔ | Единая точка доступа для всех подразделений. RBAC и групповые политики обеспечивают межкомандную координацию. RAG — общая база знаний между отделами. |
| **Тактический** | ✔ | Конкретные инструменты для ежедневной работы: AI-ассистенты, Document Intelligence, конструктор ассистентов. Сотрудники используют платформу как рабочий инструмент. |

## Для кого

* Компании, которым нужен контролируемый доступ к AI-моделям для сотрудников
* IT-подразделения, ищущие единую точку интеграции с LLM
* Организации с требованиями к безопасности данных (152-ФЗ, air-gapped контуры)
* Компании 50+ сотрудников в регулируемых отраслях (финансы, юриспруденция, медицина, госсектор)

## Что это

Собственная корпоративная ИИ-платформа Кактус.AI — полнофункциональное решение, которое даёт сотрудникам компании единый доступ к различным LLM-моделям, корпоративной базе знаний и AI-инструментам через контролируемый интерфейс. Кактус.AI берёт на себя развёртывание, настройку, поддержку и интеграцию.

## Ключевые возможности

| Возможность | Описание |
| --- | --- |
| Мультимодельный доступ | 4 провайдера (OpenAI, Anthropic, Google, локальные модели на GPU в РФ) через единый интерфейс со смарт-маршрутизацией |
| Document Intelligence | 24 инструмента для Excel/CSV, Python-песочница, парсинг PDF/DOCX/PPTX с сохранением структуры |
| AI-ассистенты | 19 готовых скиллов, конструктор собственных ассистентов, 4 MCP-сервера |
| Корпоративная база знаний (RAG) | Загрузка и индексирование документов, AI-поиск с ответами и источниками, document-level ACL |
| Enterprise-безопасность | 4-уровневый RBAC (30+ разрешений), Policy Engine (fail-closed), шифрование Fernet, аудит событий |
| K-токен экономика | Внутренняя валюта для бюджетирования AI, настраиваемые курсы per модель, лимиты по пользователям и отделам |
| Uptime | 99.95% доступность |
## Варианты развёртывания

| Вариант | Описание | Для кого |
| --- | --- | --- |
| Shared-сервер | Управляемый контур Кактус.AI, изолированные tenant-пространства | Команды до 100 человек, пилоты |
| Dedicated-сервер | Выделенный сервер, кастомные политики безопасности | Компании от 100 сотрудников |
| On-premise / свои GPU | Полная изоляция данных, air-gapped режим, локальные модели на GPU клиента | Enterprise и регулируемые отрасли |

## Аутентификация

4 провайдера: Google OAuth, OIDC, LDAP/AD, email — с поддержкой мультитенантности и групповых политик.

## Риски использования LLM в России (решаем)

| Риск | Как решаем |
| --- | --- |
| 152-ФЗ, персональные данные | Локальная LLM на GPU в РФ, смарт-маршрутизация, on-prem развёртывание |
| Блокировки и нестабильность доступа | Единый шлюз к 4 провайдерам, автоматический fallback, SLA 99.95% |
| Оплата зарубежных сервисов | K-токены с рублёвыми тарифами, закрывающие документы РФ |
| Разрозненные AI-инструменты | Единая платформа, полный аудит, гранулярное разграничение доступа |
