# Разработка AI-агентов под ключ

← [AI-направление](../обзор-ai.md) · [Карта уровней](../../../каталог-услуг.md)

> Создание кастомных AI-агентов и ассистентов для автоматизации конкретных бизнес-задач.

## На каком уровне работает

**Тактический уровень.** Создаём конкретный AI-инструмент для конкретной команды или процесса. Агент работает «в руках» у людей на тактическом уровне.

## Для кого

* Компании с конкретной задачей для автоматизации
* Подразделения без ресурсов на собственную разработку
* Компании, которые хотят быстро получить работающий прототип

## Проблема

Компании видят возможности AI-агентов (чат-боты, автоматизация документов, AI-ассистенты), но не имеют внутренней экспертизы для разработки. С развитием технологий основная работа не в коде, а в проектировании логики и обработке крайних случаев.

## Что включено

### Этап 1: Проектирование
* Определение задачи и сценариев использования
* Проектирование логики и обработки крайних случаев
* Выбор технологического стека

### Этап 2: Разработка
* Создание AI-агента / ассистента
* Подключение к данным и системам клиента (CRM, 1С и др.)
* Тестирование на реальных сценариях
### Этап 3: Внедрение
* Деплой в инфраструктуру клиента или облако
* Обучение пользователей
* Документация и передача

## Технологический стек

| Компонент | Технологии |
|---|---|
| Оркестрация | n8n, LangChain, собственные реализации |
| LLM | GPT, Claude, Google Gemini, локальные модели |
| Генерация документов | Python (docxtpl), шаблонизаторы |
| Интеграции | CRM, 1С, Telegram, корпоративные мессенджеры |
| Инфраструктура | Docker, облако или on-premise |

## Примеры агентов

| Тип | Описание |
|---|---|
| Агент формирования договоров | Автоматическая генерация договоров из CRM/1С |
| AI-ассистент для HR | Ответы на вопросы сотрудников, онбординг, FAQ |
| Агент обработки документов | Извлечение данных, классификация, маршрутизация |
| Клиентский чат-бот | Поддержка клиентов с доступом к базе знаний |
| AI-ассистент для Scrum-команды | Автоматизация ретроспектив, трекинг блокеров |
| Аналитический агент | Автоматизация отчётов и анализа данных |

## Длительность

2–6 недель (~160 часов)

## Результат

* Работающий AI-агент, интегрированный в процессы
* Документация и инструкции по поддержке
* Возможность доработки и масштабирования
