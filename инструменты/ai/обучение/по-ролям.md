# Обучение AI по ролям

← [AI-направление](обзор-ai.md) · [Карта уровней](каталог-услуг.md)

> Программы, адаптированные под конкретные должности — не абстрактные знания, а AI для вашей ежедневной работы.

## На каком уровне работает

**Тактический уровень.** Работает с группой одной роли — Team Lead, продакт-менеджеры, аналитики, HR, маркетологи, IT. Каждая минута обучения релевантна ежедневным задачам участника.

## Для кого

| Роль | Фокус |
|---|---|
| Team Lead / руководители | AI в управлении командой, приоритизации, коммуникациях |
| Продакт-менеджеры | AI для исследований, анализа данных, генерации гипотез |
| Аналитики | AI для работы с данными, визуализации, автоматизации отчётов |
| HR / T&D | AI для рекрутинга, оценки, развития сотрудников |
| Маркетологи | AI для контента, аналитики, персонализации |
| IT-специалисты | AI-инструменты для разработки, тестирования, DevOps |

## Проблема

Общие программы по AI дают широкую картину, но участники часто не понимают, как конкретно применять это в своей роли. Программы по ролям закрывают этот разрыв — каждая минута обучения релевантна ежедневным задачам участника.

## Что включено

- Аудит текущих процессов роли: где AI создаёт максимальную ценность
- Подбор инструментов под конкретные задачи роли
- Практика на реальных рабочих кейсах участников
- Шаблоны промптов и workflow для типичных задач роли
- План внедрения AI в рабочую рутину

## Форматы

| Формат | Описание |
|---|---|
| Корпоративный | Группа одной роли из компании (10–20 человек) |
| Открытый | Сборная группа одной роли из разных компаний |
| Workshop | Короткий интенсив (полдня) для quick win |

## Длительность

0.5–2 дня (зависит от глубины проработки)

## Результат

- Каждый участник уходит с конкретным набором AI-инструментов для своей роли
- Готовые шаблоны и workflow для немедленного применения
- Измеримое ускорение типичных задач
