# AI-тренажёры: Нота.Кактус

← [AI-направление](обзор-ai.md) · [Карта уровней](каталог-услуг.md)

> Массовое обучение сотрудников через AI-бот тренажёр — сценарное обучение в формате диалога.

## На каком уровне работает

**Тактический уровень.** Инструмент масштабного обучения, работает индивидуально с каждым сотрудником через AI-бота.

## Для кого

- Компании, которым нужно быстро и недорого обучить большой поток сотрудников
- HR/T&D-отделы, ищущие масштабируемые инструменты обучения
- Дополнение к любой обучающей программе для закрепления навыков

## Проблема

Тренд 2025–2026: компании хотят обучить большой поток быстро и за разумный бюджет. Классические тренинги не масштабируются линейно. Нужен инструмент, который работает 24/7, адаптируется под каждого участника и даёт измеримый результат.

## Продукт: Нота.Кактус

AI-бот тренажёр для сценарного обучения. Работает как чат-бот (Telegram или интеграция в корпоративный мессенджер/LMS).

### Как это работает

1. Сотрудник открывает чат-бот
2. Бот ведёт сценарный диалог: ставит задачи, даёт контекст, оценивает ответы
3. AI адаптирует сложность под уровень участника
4. По итогам — аналитика: что усвоено, где пробелы

### Готовые курсы

| Курс | Описание |
|---|---|
| Prompt Engineering | Основы промпт-инжиниринга через практику в диалоге с ботом |
| Кастомные сценарии | Разрабатываем сценарии под процессы и задачи вашей компании |

### Возможности

- Сценарное обучение с ветвлением диалога
- Адаптивная сложность
- Дашборд для HR: прогресс, оценки, аналитика по каждому сотруднику
- Интеграция с корпоративной LMS
- Работа через Telegram или веб-интерфейс

## Варианты поставки

| Вариант | Описание |
|---|---|
| Базовый (чат-бот) | Чат-бот с готовыми или кастомными сценариями |
| С интеграцией в LMS | Бот + подключение к корпоративной системе обучения, расширенная аналитика |

## Длительность

Доступ на 1–6 месяцев. Каждая сессия — 15–30 минут.

## Результат

- Масштабируемое обучение без ограничений по количеству участников
- Измеримая оценка навыков каждого сотрудника
- Снижение стоимости обучения на одного участника при масштабировании
