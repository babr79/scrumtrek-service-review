# ScrumTrek — Каталог услуг

> Образовательно-консалтинговый холдинг с 2007 года. Помогаем бизнесу достигать целей, выпускать крутые продукты и работать осознанно — без лишней бюрократии.

## О компании

ScrumTrek начинал как небольшая коучинговая компания, которая одной из первых поверила в Agile и начала продвигать его в России. Сейчас ScrumTrek помогает бизнесу решать проблемы и повышать эффективность на всех уровнях: от топ-менеджмента до линейных разработчиков.

**Основатель:** Асхат Уразбаев (2007 год).

---

## Модель: Уровни × Инструменты

В основе всего, что делает ScrumTrek, лежит **матричная модель**. Она состоит из двух измерений:

- **Три уровня** — стратегический, координационный, тактический — это уровни принятия решений в организации. У каждого свои задачи, ритм и аудитория.
- **Четыре инструмента** — Стратегатор, Agile-практики, AI / Кактус.AI, Инженерная культура — это наборы компетенций и методологий. Каждый инструмент работает на всех трёх уровнях, но по-разному.

Услуги ScrumTrek живут на пересечении уровня и инструмента. Уровни — это не отделы и не продукты, а измерение, которое пронизывает все направления.

```

                    │             │  практики │ Кактус.AI │  культура  │
┌───────────────────┼─────────────┼───────────┼───────────┼────────────┤
│ СТРАТЕГИЧЕСКИЙ    │ Стратегия   │ Business  │ AI-       │ IT-        │
│ топ-менеджмент,   │ Портфель    │ Agility   │ стратегия │ диагности- │
│ C-level           │ OKR         │ LPM       │ AI Audit  │ ка, Event  │
│                   │ Инновации   │ EAC       │ DDO       │ Storming   │
├───────────────────┼─────────────┼───────────┼───────────┼────────────┤
│ КООРДИНАЦИОННЫЙ   │ Value       │ SAFe      │ AI-центр  │ Карта      │

├───────────────────┼─────────────┼───────────┼───────────┼────────────┤
```

### Формат описания каждой услуги

Каждый файл услуги содержит:

1. **Что это** — полное описание без сжатия
2. **Для кого** — целевая аудитория
3. **Проблема** — какую боль решает
4. **На каком уровне работает** — привязка к стратегическому / координационному / тактическому
5. **Программа / Что включено** — детальное содержание
6. **Форматы и длительность**
7. **Результат**
8. **Кейсы** — с деталями
9. **Связи** — перекрёстные ссылки на другие услуги
